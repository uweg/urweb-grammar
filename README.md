# urweb-grammar

This package provides a grammar for Ur/Web, which could be used to create plugins for various editors. Currently, it only supports `.ur` files.

The grammars are created with [iro](https://eeyo.io/iro/). The files in the `generated` folder are generated and should not be edited manually.

## Used in
This package is currently used in the following plugin:

* [vscode-urweb](https://github.com/uweg/vscode-urweb)

If you have any additions to this list, feel free to create a pull request.
